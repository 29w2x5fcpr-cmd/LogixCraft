# Roadmap

This roadmap is intentionally lightweight. It records direction without pretending that every feature is already designed.

## Foundation

- Application shell and navigation.
- Theme and font system.
- Preferences dialog.
- Startup validation.
- Logging and log rotation.
- Global error handling.
- Basic automated test runner.
- Splash, About, License, Help Viewer, Terminal dialogs.

## Near-Term Work

- Improve main window layout and navigation ergonomics.
- Expand Help Viewer into real documentation pages.
- Add user-facing settings export/import.
- Add clearer release/version workflow.
- Add packaging notes for Windows builds.
- Improve startup diagnostics and recovery options.

## Engineering Tool Modules

Planned module areas:

- PLC tools.
- HMI/SCADA utilities.
- Industrial network tools.
- Commissioning workflows.
- Diagnostics and troubleshooting helpers.
- Engineering calculations.
- Documentation and report generation.

## Release Work

- Automated version bumping.
- Git tag based release process.
- Windows packaging.
- Installer or portable build.
- Release notes generation.
