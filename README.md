LogixCraft

Professional Toolkit for Rockwell PLC Development and Commissioning

LogixCraft is a Python-based software suite designed for engineers working with Allen Bradley ControlLogix and CompactLogix systems. It provides tools for development, commissioning, diagnostics, and process optimization in industrial environments.

⸻

Features (Planned and In Progress)

PLC Integration
	•	Live connection to Rockwell PLCs
	•	Real-time tag read and write
	•	Support for:
	•	Kp, Ki, Kd tuning
	•	Setpoint (SP), Process Variable (PV), Control Variable (CV)

PID Tuning Module
	•	Closed-loop tuning using live feedback
	•	Integration with custom pyPIDTune
	•	Real-time trend visualization
	•	Safe tuning workflow

Engineering Utilities
	•	L5K file parsing and modification
	•	Tag analysis tools
	•	Commissioning utilities
	•	Troubleshooting support

GUI Interface
	•	Built with PySide6 (Qt)
	•	Modular and scalable architecture
	•	Designed for engineering workflows

Integrated Help System (Planned)
	•	In-application documentation viewer
	•	Searchable help content
	•	Commissioning and troubleshooting guides
	•	Bookmarking and tagging system

Licensing System (Planned)
	•	Software activation keys
	•	Hardware-based licensing
	•	Anti-piracy protection
